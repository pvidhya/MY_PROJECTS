*************BESTDEAL***********
***Web-based application for selling electronic goods where Customers, Salemen and the Store Managers have different functionality.

****HTML, CSS, JavaScript, JSP, Servlets, Ajax, MySql, MongoDB, Python, Twitter API

1. All the .java files are in source.
2. Compile them using javac filename.java
3. Save all the class files in class folder.
4. Go to bin and start the server.
5. Start MySQL server.
6. Start mongod and mongo server.
7. Run the application with local host.
